# Codex

A lightweight Markdown codex for the Serpent Bloom project.

## Structure

- **Overview**: High-level description and intent.
- **Decisions**: Record notable choices and their rationale.
- **Todos**: Track follow-up work or open questions.

## Usage

Add new entries as Markdown sections. Keep them concise, dated when helpful, and link to related resources or issues for context.
